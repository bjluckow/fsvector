# fsvector v1.4 — Video Processing

## Overview

Adds video file support. A video produces two types of rows:
- **Frames** — sampled at `VIDEO_FRAME_RATE` fps, embedded as images via CLIP
- **Transcript** — audio track extracted, transcribed via Whisper, chunked
  and embedded as text

Both row types share the same `path`. Transcript chunks are offset by the
frame count to avoid `chunk_index` collisions:
```
video.mp4, chunk_index=0..N-1,   chunk_type='frame'
video.mp4, chunk_index=N..N+M-1, chunk_type='transcript'
```

v1.4 also refactors `convertsvc` from a single multiplexed `/convert`
endpoint into focused per-modality endpoints, and introduces typed methods
on `internal/convert/client.go`.

## M1.4.1 — convertsvc endpoint refactor

### New convertsvc API
```
GET  /health
POST /convert/text    (file, target_format)   → plain text bytes
POST /convert/image   (file, target_format)   → image bytes
POST /convert/audio   (file)                  → 16kHz mono WAV bytes
POST /convert/video/audio  (file)             → 16kHz mono WAV bytes
POST /convert/video/frames (file, fps)        → multipart JPEG frames
```

`/convert/video/frames` returns a `multipart/mixed` response where each
part is a JPEG frame with headers:
```
Content-Type: image/jpeg
X-Frame-Index: 0
X-Timestamp-Ms: 0
```

### convertsvc/main.py structure
```python
@app.post("/convert/text")
async def convert_text(file: UploadFile, target_format: str = Form(...)):
    # routes to pdftotext or pandoc

@app.post("/convert/image")
async def convert_image(file: UploadFile, target_format: str = Form(...)):
    # routes to imagemagick

@app.post("/convert/audio")
async def convert_audio(file: UploadFile):
    # normalizes to 16kHz mono WAV via ffmpeg

@app.post("/convert/video/audio")
async def convert_video_audio(file: UploadFile):
    # extracts + normalizes audio track via ffmpeg
    # reuses _normalize_audio internally

@app.post("/convert/video/frames")
async def convert_video_frames(file: UploadFile, fps: float = Form(1.0)):
    # extracts frames via ffmpeg
    # returns multipart/mixed response
```

Old `/convert` endpoint removed.

### internal/convert/client.go — typed methods

## M1.4.2 — video file types + config

### mappings.go
```go
// video — always needs processing
"mp4": {"video", ""},
"mov": {"video", ""},
"avi": {"video", ""},
"mkv": {"video", ""},
```

`ConvertTarget` returns `""` for video — conversion is handled
differently per output type (frames vs audio) so pipeline owns
the routing explicitly.

### config.go
```go
VideoFrameRate float64  // from VIDEO_FRAME_RATE, default 1.0
```
```go
VideoFrameRate: envFloat("VIDEO_FRAME_RATE", 1.0),
```

Add `envFloat` helper to config.go:
```go
func envFloat(key string, fallback float64) float64 {
    v := os.Getenv(key)
    if v == "" {
        return fallback
    }
    f, err := strconv.ParseFloat(v, 64)
    if err != nil {
        return fallback
    }
    return f
}
```

### .env.example
```
VIDEO_FRAME_RATE=1.0
```

### pipeline.Config
```go

```

## M1.4.3 — processVideo

### pipeline.go
```go
func processVideo(ctx context.Context, cfg Config, fi fsindex.FileInfo) (Result, error) {
    data, err := readFile(fi.Path)
    if err != nil {
        return Result{}, fmt.Errorf("read %s: %w", fi.Path, err)
    }

    var files []store.File
    frameType := "frame"
    transcriptType := "transcript"

    // 1. extract and embed frames
    frames, err := cfg.ConvertClient.ExtractVideoFrames(ctx, fi.Name, data, cfg.VideoFrameRate)
    if err != nil {
        // non-fatal — log and continue to transcript
        fmt.Fprintf(os.Stderr, "    extract frames %s: %v\n", fi.Path, err)
    } else {
        for _, frame := range frames {
            vector, err := cfg.EmbedClient.EmbedImage(ctx, fi.Name, frame.Data)
            if err != nil {
                continue
            }
            files = append(files, store.File{
                Path:           fi.Path,
                Source:         cfg.Source,
                ContentHash:    fi.Hash,
                Size:           fi.Size,
                MimeType:       fi.MimeType,
                Modality:       "video",
                FileName:       fi.Name,
                FileExt:        fi.Ext,
                FileCreatedAt:  &fi.CreatedAt,
                FileModifiedAt: &fi.ModifiedAt,
                EmbedModel:     cfg.EmbedModel,
                Embedding:      vector,
                ChunkIndex:     frame.Index,
                ChunkType:      &frameType,
                Metadata: map[string]any{
                    "timestamp_ms": frame.TimestampMs,
                    "frame_index":  frame.Index,
                    "fps":          cfg.VideoFrameRate,
                },
            })
        }
    }

    transcriptOffset := len(frames)

    // 2. extract audio and transcribe
    audioData, err := cfg.ConvertClient.ExtractVideoAudio(ctx, fi.Name, data)
    if err != nil {
        fmt.Fprintf(os.Stderr, "    extract audio %s: %v\n", fi.Path, err)
    } else {
        resp, err := cfg.TranscribeClient.Transcribe(ctx, fi.Name+".wav", audioData)
        if err != nil {
            fmt.Fprintf(os.Stderr, "    transcribe %s: %v\n", fi.Path, err)
        } else if strings.TrimSpace(resp.Text) != "" {
            chunks := chunk.Split(resp.Text, cfg.ChunkSize, cfg.ChunkOverlap, cfg.MinChunkSize)
            for i, c := range chunks {
                f, err := processTextChunk(ctx, cfg, fi, c, transcriptOffset+i)
                if err != nil {
                    continue
                }
                if f != nil {
                    f.Modality = "video"
                    f.ChunkType = &transcriptType
                    f.Metadata = map[string]any{
                        "duration_seconds": resp.DurationSeconds,
                        "language":         resp.Language,
                    }
                    files = append(files, *f)
                }
            }
        }
    }

    if len(files) == 0 {
        return Result{
            Skipped:    true,
            SkipReason: "no frames or transcript produced",
        }, nil
    }

    return Result{Files: files}, nil
}
```

Wire into `Process()`:
```go
case "video":
    return processVideo(ctx, cfg, fi)
```
